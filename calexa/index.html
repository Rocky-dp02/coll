Enable Proxy Mode (Fixes 403 Forbidden)

Why is it saying 403 Forbidden?

Modern browsers block custom headers (like the Referer) for security when you host this on your own domain. To fix this:

  • Toggle "Enable Proxy Mode" above. This uses AllOrigins to bypass browser security.
  • Alternatively, install a CORS Unblock extension in your browser.
  • If the Proxy fails, it means the S2S server blocks all external IP addresses entirely, and you may need to use a VPN or run this from within the S2S network.
